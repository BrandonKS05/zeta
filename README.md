# treehacks-2026 monorepo starter

This repo is set up for:

- A Modal-hosted translation backend using `FrenzyMath/Herald_translator`
- A Chrome extension frontend that runs only on Overleaf and highlights target words in the source editor

## Layout

```text
apps/
  overleaf-extension/
services/
  translator-modal/
```

## Quick start

1. Set up and deploy Modal backend:
   - See `services/translator-modal/README.md`
2. Load Chrome extension:
   - See `apps/overleaf-extension/README.md`
